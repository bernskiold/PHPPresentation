# 1.3.0

[Full Changelog](https://github.com/PHPOffice/PHPPresentation/compare/1.2.0...1.3.0)

## Enhancements
- `phpoffice/phpspreadsheet`: Allow version 5.0 by [@seanlynchwv](http://github.com/seanlynchwv) in [#879](https://github.com/PHPOffice/PHPPresentation/pull/879)
- New `AdvancedScatter` chart type with proper numeric X / Y axes, multiple series, duplicate X values across data points, configurable scatter style (`marker`, `lineMarker`, `smooth`, etc.), per-data-point coloring and `varyColors` support — matching PowerPoint's scatterChart capabilities. Read and write supported for the PowerPoint 2007 (.pptx) format.
